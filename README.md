# Go Offer Docs

Генератор документов по шаблонам. Несколько типов (Official Freeze Form,
Extension of Program Term + свои), у каждого: создание по шагам, редактируемый
текстовый шаблон с форматированием, список созданных документов.

## Файлы
| Файл | Назначение |
|------|------------|
| `app.py` | Приложение (Streamlit) |
| `engine.py` | Генерация .docx (разметка + таблица подписей), расчёты, PDF |
| `editor.py` | Встроенный редактор шаблона (форматирование + вставка переменных) |
| `requirements.txt` | Python-зависимости |
| `packages.txt` | LibreOffice для PDF (в КОРНЕ репозитория) |
| `data.json` | Создаётся автоматически (шаблоны, переменные, типы, документы) |

## Возможности
- Меню — сворачиваемые разделы (тоглы) по типам документов.
- Глобальные переменные: `{name}` = имя клиента везде; при добавлении поля
  выбираешь переменную из списка или создаёшь новую.
- Шаблон — единый текст с разметкой `**жирный**`, `*курсив*`, `# Заголовок`,
  `## Подзаголовок`. Кнопки форматирования и кликабельные переменные (вставка по курсору).
- Таблица подписей добавляется автоматически (широкие поля, одинакова во всех документах).

## Запуск локально
```bash
pip install -r requirements.txt
streamlit run app.py
```
Для PDF нужен LibreOffice.

## Деплой (Streamlit Community Cloud)
`packages.txt` (строка `libreoffice`) — в КОРНЕ репозитория. Main file path → путь к `app.py`.
